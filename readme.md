# go-edi - edifact reader written in golang

### WIP!

## Features

* low level streaming
* generic object model, dom
* supports UN/EDIFACT

## Unsupported Features/ Limitations

* dom does not support repetition seperators

## Planned Features

* writing
* message type specific object models
